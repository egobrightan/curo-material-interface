Flex Modal

Modal with following specs:

  • Fixed
  • 90% screen width (5% left/right margin)
  • max. 90% height (5% top margin, min. 5% bottom margin)
  • Header/Footer always visible
  • Content overflow scroll
Open example modal 
.cuma.cuma-flex-modal
  .cuma-flex-modal-dialog
    [.cuma-flex-modal-dialog-inner] // optionally if 2nd container is necessary
      %header
        .cuma-flex-modal-header-inside
          .cuma-pull-left
            %a.cuma-flex-modal-back.btn.ripples{ href: '#' }
              %i.mdi-navigation-arrow-back
          .cuma-pull-right
            %a.cuma-flex-modal-close.btn.ripples{ href: '#' }
              %i.mdi-navigation-close
          %h1 Modal

      %main
        .cuma-flex-modal-main-inside
          %aside
            %p My Sidebar
          %section
            %p My Content
      %footer
        .cuma-flex-modal-footer-inside
          .cuma-pull-left
            %a.cuma-flex-modal-back.btn.ripples{ href: '#' }
              %i.mdi-navigation-arrow-back
          .cuma-pull-right
            %a.cuma-flex-modal-close.btn.btn-default.ripples.ripples-dark{ href: '#' }
              Close
            %a.cuma-flex-modal-save.btn.btn.btn-raised.btn-primary.ripples{ href: '#' }
              Save

To preseve the content from scrolling when the modal is opened, you should add the following class to the body tag: 

body.cuma-flex-modal-open
